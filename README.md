# Mikir ( •_•)

Selamat datang di **Mikir**, aplikasi kuis yang nggak kaku dan penuh dengan wajah-wajah lucu (kaomoji).

Bosen sama aplikasi belajar yang serius amat? Sama. Makanya ini dibuat. ( ◡ _ ◡ )

---

## Fitur-Fitur Aneh (Tapi Keren)

### 1. AI yang Rajin Baca ( ✧ ▽ ✧ )
Kamu punya PDF numpuk? Slide kuliah yang nggak pernah dibaca? Upload aja ke sini. Biar AI yang baca, terus dia bikinin kuis buat kamu. Kamu tinggal **Mikir** jawabannya.

### 2. Kendalikan Pakai Hidung ( ⊙ _ ⊙ )
Males nyentuh layar atau mouse? Aktifin **Eye Tracking**.
- Gerakin hidungmu buat arahin kursor.
- Tahan (dwell) di atas jawaban buat milih.
- Angguk-angguk atau geleng-geleng (eh nggak deng, cuma arahin aja).

### 3. Jedi Mode (Hand Gestures) ( ✋ ◡ ✋ )
Lagi makan gorengan dan tangan berminyak? Jangan pegang HP/Keyboard!
- Angkat tanganmu ke kamera.
- **Cubit (Pinch)** jari telunjuk & jempol buat milih jawaban.
- **Kepal Tangan** buat scroll atau aksi lainnya.
- Berasa jadi penyihir.

### 4. Kaomoji yang Ekspresif
- Kalau bener: `( ✧ ▽ ✧ )` "Mantap!"
- Kalau salah: `( ≧Д≦)` "Waduh..."
- Kalau mikir: `( . _ . )?` "Hmm..."
- Kalau streak: `( 🔥 ◡ 🔥 )` "ON FIRE!"

---

## Cara Pakai (Gampang Kok)

1. **Siapin API Key**: Pake Google Gemini (Gratis kok) atau Groq kalau mau ngebut.
2. **Upload Materi**: PDF, Teks, terserah.
3. **Pilih Mode**:
   - **Standard**: Santai.
   - **Survival**: Salah 3x = Game Over. ( ; ω ; )
   - **Time Rush**: Balapan sama waktu.
4. **Mainkan!**
