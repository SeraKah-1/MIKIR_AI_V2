# Contributing to Mikir ( •_•)

Terima kasih telah tertarik untuk berkontribusi pada Mikir! Kami sangat menghargai bantuan Anda untuk membuat aplikasi ini menjadi lebih baik.

## Cara Berkontribusi

1.  **Laporkan Bug**: Jika Anda menemukan bug, silakan buat Issue baru dengan deskripsi yang jelas tentang cara mereproduksi bug tersebut.
2.  **Saran Fitur**: Kami selalu terbuka untuk ide-ide baru! Silakan buat Issue untuk mendiskusikan fitur yang ingin Anda tambahkan.
3.  **Pull Requests**:
    *   Fork repositori ini.
    *   Buat branch baru untuk fitur atau perbaikan bug Anda.
    *   Pastikan kode Anda mengikuti standar yang ada dan lulus pengujian (`npm run lint` dan `npm run test:unit`).
    *   Kirimkan Pull Request dengan deskripsi yang jelas tentang perubahan Anda.

## Standar Kode

*   Gunakan TypeScript untuk semua kode baru.
*   Ikuti pola komponen fungsional React dan Hooks.
*   Gunakan Tailwind CSS untuk styling.
*   Pastikan untuk menambahkan komentar pada logika yang kompleks.

## Lisensi

Dengan berkontribusi pada repositori ini, Anda setuju bahwa kontribusi Anda akan dilisensikan di bawah Lisensi MIT.
